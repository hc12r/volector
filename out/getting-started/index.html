Getting Started

Quick setup guide to get Volector running in your environment

Step 1Installation
Install Volector using pip with your preferred dependency set

Basic Installation

bash
1pip install volector

With Rendering Support

bash
1pip install volector[render]

Full Installation (All Features)

bash
1pip install volector[all]
Step 2Environment Setup
Configure MinIO and other optional services
bash
1# MinIO Configuration (Optional)
2export MINIO_ENDPOINT=localhost:9000
3export MINIO_ACCESS_KEY=your-access-key
4export MINIO_SECRET_KEY=your-secret-key
5export MINIO_BUCKET=volector-data
6
7# Rendering Configuration (Optional)
8python -m playwright install chromium
Step 3First Crawl
Run your first web crawl with Volector

Quick Test

bash
1volector run example-news --country MZ --max-pages 5

With Raw Output

bash
1volector run example-news --country MZ --write-raw --max-pages 10
Step 4Configuration
Set up your sources configuration file

Create crawler/config/sources.yaml:

yaml
1sources:
2  - name: example
3    base_urls: ["https
4    country: MZ
5    language: pt
6    render: false
7    allow: ["/news/"]
8    deny: ["/login", "/privacy"]
9    schedule: "0 * * * *"   # hourly
10    max_pages: 200
